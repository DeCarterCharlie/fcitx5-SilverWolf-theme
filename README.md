# 安装
<p>解压下载到的tar.gz包并复制主题目录到 ~/.local/share/fcitx5/themes/ 目录下。</p>
如：

![安装](屏幕截图_20241224_013048.png)

<p>打开 fcitx5配置，选择 配置附加组件，经典用户界面栏点击配置，选择对应主题即可。</p>
